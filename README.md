# ChemicalApparatus-
App for calculation scheme with chemical apparatuses by 0d models

## Решатель аппаратов (веб)

`Решатель_аппаратов.html` — самодостаточный браузерный решатель 0d-моделей
(один файл, без установки; графики — Plotly). Тот же файл лежит и внутри
`Химаппараты.zip` (`Химаппараты/Решатель_аппаратов.html`).

Режимы:

- **Аппараты · параметры/сенсоры/тесты** — для каждого из 4 аппаратов
  **CHE / FHE / EOH / HRU** показан полный набор **параметров** и **сенсоров**,
  взятый напрямую из C++-библиотеки `perm-devices` (`devices/*.cpp`:
  `m_parameters`, `m_sensors`). Все **12 тестовых режимов** из
  `Тесты для аппаратов/*/test.json` перенесены в HTML, граничные условия — из
  `config.xml`; режим загружается одним кликом.
  **Физика перенесена из C++** (`devices/`, `elements/`): по кнопке «Рассчитать»
  модель считает теплообмен по всей геометрии аппарата —
  **коэффициенты теплоотдачи α вычисляются из корреляций Re/Nu/Pr (не задаются
  вручную)**, тепло идёт **газ→масло** через UA (α_газ + стенка + α_масло) и ε-NTU;
  свойства — фиты Therminol-66 (масло) и продуктов сгорания (газ). FHE учитывает
  горение CH₄ (ограничено O₂) и радиацию; EOH — динамику пучка ТЭН и расход масла
  из перепада давлений. Все сенсоры (T газа/масла на входе/выходе, α, LMTD, Q,
  расходы, ΔP…) заполняются вычисленными значениями и сверяются с эталоном режима.
- **Связка** использует те же полные модели аппаратов, что и вкладка «Аппараты»:
  каждая станция — CHE/FHE/HRU/EOH с полным набором параметров (⚙, загрузка режима),
  гранусловиями газа и **выводом всех сенсоров**; список станций общий для обеих топологий.
  - **Открытая цепочка** — масло проходит станции один раз (sequential-modular).
- **Связка → Замкнутый контур масла** — масло циркулирует через насос и резервуар
  по цепочке станций:
  **Резервуар → Насос → Утилизатор HRU → Печной FHE → Огневой EOH → Потребитель → Резервуар**.
  Нагреватели **HRU** и **FHE** греют масло дымовым газом (газ после отдачи тепла
  **выбрасывается в атмосферу**), **EOH** добавляет тепло электроподогревом, а
  **Потребитель** отбирает тепло на технологический процесс.
  Список станций **полностью настраиваемый** (как в открытой цепочке): станции
  можно добавлять, удалять и переставлять (▲▼); типы:
  - **Печь-утилизатор** — смеситель воздуха и метана → печь **идеального
    (полного) сгорания** CH₄ (CH₄+2O₂→CO₂+2H₂O, L₀≈17.2, LHV≈50 МДж/кг) →
    дымовые газы (масса и адиабатическая температура пламени считаются из баланса,
    зависят от расхода CH₄ и коэффициента избытка воздуха α) → утилизатор тепла
    (ε-NTU, газ→масло) → **выброс в атмосферу**;
  - HRU/FHE/CHE (газ→масло, температура и расход газа задаются напрямую),
    EOH (электро), Потребитель (масло→процесс).
  Это задача с рециклом: стационар ищется разрывом контура tear-потоком и
  под-релаксацией (§2.5 рекомендаций), равновесие — баланс
  **подвод тепла (нагреватели) + насос = отбор потребителя + потери в среду**.
  Энергобаланс контура замыкается точно; неявный и явный Эйлер сходятся к одному
  стационару (явный при крупном Δt теряет устойчивость). Строятся: схема контура,
  **профиль температуры масла по контуру**, кривая выхода на режим и график
  сходимости рецикла.

- **Свойства потоков** — температурные зависимости всех свойств потоков
  (плотность, теплоёмкость, теплопроводность, динамическая вязкость) и молярная
  масса. Масло — **THERMINOL 59** (фиты калиброваны по паспорту L3-87Y02, реж.710:
  паспортные точки 185.06/262 °C показаны на графиках и лежат на кривых); газ — **дымовые газы на УТ** (фиты по замерам в СИ: ρ=346/T, M≈28.4;
  cp/λ/μ линейные; замеренные точки показаны на графиках). HRU-тесты приведены к паспорту: масло 1 066 292 кг/ч
  (296.19 кг/с), 185→262 °C, и полный поток газа даёт Q≈53647 кВт.
